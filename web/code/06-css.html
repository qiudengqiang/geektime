公司需求岗位

软件工程师
招聘人数:3-8人   月薪：2000-6000元
岗位职责：
1. 根据产品需求进行产品设计、开发和测试；
2. 编写产品相关技术文档；
3. 为产品实施团队提供技术支持。
任职要求：
1.计算机专业本科学历
2.具有强烈的责任心、工作认真踏实、有良好的沟通能力和团队合作精神、承压能力
3.5年及以上J2EE开发经验；能完成系统的设计、开发、测试等工作
4.熟练掌握SSH，SpringMVC等主流框架
5.熟练使用Eclipse、SVN等工具
6.熟悉前端技术Jsp,Servlet,Javascript,AJAX,jquery包括HTML/CSS/JS/XML的开发。
7. 熟练掌握Oracle、MS SQLServer、MySQL等主流数据库，熟悉数据库建模，具备深厚的SQL功底，并有着丰富的数据库设计经验；
8. 具有系统需求分析和设计能力，能快速理解业务需求并进行相关逻辑设计，熟悉UML建模，并熟练使用相关建模工具，并具有规范的开发、设计文档写作能力
